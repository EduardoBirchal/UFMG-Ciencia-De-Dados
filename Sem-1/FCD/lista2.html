3. Definindo Funes

Vamos escrever uma funo muito simples que converte uma proporo em uma porcentagem multiplicando-a por 100. Por exemplo, o valor de to_percentage(0.5) deve ser o nmero 50. (Sem sinal de porcentagem).

Uma definio de funo possui algumas partes.

def

Sempre comea com def (abreviao de define):

def

Nome

Em seguida vem o nome da funo. Vamos chamar nossa funo de to_percentage.

def to_percentage

Assinatura

Em seguida vem algo chamado assinatura da funo. Isso informa ao Python quantos argumentos sua funo deve ter e quais nomes voc usar para se referir a esses argumentos no cdigo da funo. to_percentage deve receber um argumento, e chamaremos esse argumento de proporo, pois deve ser uma proporo.

def to_percentage(proportion)

Colocamos dois pontos aps a assinatura para informar ao Python que acabou.

def to_percentage(proportion):
Documentao

Funes podem fazer coisas complicadas, ento voc deve escrever uma explicao do que sua funo faz. Para funes pequenas, isso menos importante, mas um bom hbito aprender desde o incio. Convencionalmente, as funes Python so documentadas escrevendo uma string entre aspas triplas:

def to_percentage(proportion):
"""Converte uma proporo em porcentagem."""
Corpo

Agora comeamos a escrever o cdigo que executado quando a funo chamada. Isso chamado de corpo da funo. Podemos escrever qualquer coisa que escreveramos em qualquer outro lugar. Primeiro, vamos dar um nome ao nmero pelo qual multiplicamos uma proporo para obter uma porcentagem.

def to_percentage(proportion):
"""Converte uma proporo em porcentagem."""
fator = 100
return

A instruo especial return no corpo de uma funo diz ao Python para tornar o valor da chamada de funo igual ao que vem logo aps return. A instruo return tambm termina a funo e uma vez chamada nenhuma linha no corpo aps ser executada. Ento, queremos que o valor de to_percentage(0.5) seja a proporo 0.5 vezes o fator 100, ento escrevemos:

def to_percentage(proportion):
"""Converte uma proporo em porcentagem."""
fator = 100
return proportion * fator

Tudo bem, de volta aos nossos CEOs. Confira a coluna '% Change' em compensation. Mostra o aumento percentual na remunerao do CEO em relao ao ano anterior. Para CEOs sem nenhum ano anterior registrado, em vez disso diz '(No previous year)'. Os valores nesta coluna so strings, no nmeros, portanto, assim como a coluna 'Total Pay', ela no pode ser usada sem um pouco de trabalho extra.

Dado o seu salrio atual e o aumento percentual em relao ao ano anterior, voc pode calcular o salrio do ano anterior. O salrio deste ano e o salrio do ano passado esto relacionados pela frmula:

$$\text{salrio deste ano} = \text{salrio do ano passado} \cdot \left(1+\dfrac{\text{aumento percentual}}{100}\right)$$

Por exemplo, se voc ganhou $600 no ano passado e seu salrio aumentou 50%, ento

$$\begin{align} \text{pagamento deste ano} &= \text{600} \cdot \left(1+\dfrac{\text{50}}{100}\right)\\ &= \text{600} \cdot (1.5) \\ &= 900 \end{align}$$

Como sabemos como o salrio deste ano e o salrio do ano passado esto matematicamente relacionados, podemos resolver esta equao para o salrio do ano passado para fornecer uma frmula para o salrio do ano anterior se conhecermos o salrio do ano atual e o aumento percentual do ano anterior para o ano atual. Fazendo um pouco de lgebra, encontramos:

$$\text{salrio do ano passado} = \dfrac{\text{salrio deste ano}}{\left(1+\dfrac{\text{aumento percentual}}{100}\right)}$$
